## Anti-Patterns Verdict

LLM: Borderline — token system confident, inner-page execution has AI fingerprint. Four inner pages open with identical masthead structure. --color-accent = --color-error (same hex). Italic device becomes a template slot. Detector: Exit 0, zero findings — codebase passes all automated checks cleanly.

## Overall Impression

Tokens are right. Components are coherent. Home does its job. But every inner page feels like an admin panel in Japandi tokens. The single biggest opportunity: every inner page needs its own opening gesture. The availability page should feel like a building directory. Contact should feel like a sales desk. Right now they all feel like the same email in different typefaces.

## What's Working

1. Token system is genuinely confident — two palettes, named max-widths, near-zero radius scale, three fonts with clear jobs.
2. Mongolian-first consistently — aria-labels, price formatting, Golos Text with cyrillic-ext.
3. Three-state pinned image (morning/evening/green) is the smartest idea in the codebase — the one moment the site actually achieves "feel the space."

## Priority Issues

[P0] Accent-color collision destroys hierarchy — --color-accent = --color-error = same hex, used for italics, CTAs, and errors simultaneously. Fix: oak for CTAs/emphasis, terracotta for errors only.

[P0] Every inner page masthead is interchangeable — same Eyebrow + EditorialHeading + muted paragraph at same padding across 4 pages. Fix: each page needs its own opening gesture.

[P1] Availability is a spreadsheet, not a sales tool — 6-column table with 12px mono, no per-unit visual, no sort, orientation as 2-letter code. Fix: card-per-unit with oversize price, compass diagram, sticky filters.

[P1] Contact form success state is cold — two centered lines, no timeframe, no phone channel, no next step. Fix: "Бид 24 цагийн дотор залгана" + direct tel: link + brochure download.

[P2] Reveal motion is uniform — every element: y:14, opacity:0, duration:0.65 identically. Fix: 3 motion primitives — RevealHeading, RevealImage, RevealStat.

[P2] Mobile nav overlay is dark and instant on a light brand — instant black wall over cream, no animation, no active-route indicator. Fix: 200ms ease-in, cream bg, oak active state.
